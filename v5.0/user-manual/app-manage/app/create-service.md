## 创建服务组件说明

在拓扑图中创建的服务组件只可以创建在当前应用中，如果想在其他应用中创建，可在创建应用中去创建，也可在拓扑图中创建后修改该服务的所属应用；在拓扑图中创建的服务默认是创建并部署的，创建后会在拓扑图中会实时更新，如果选择创建不部署的操作，创建服务后会跳转到服务详情页面，可对当前的服务进行设置后再部署。
