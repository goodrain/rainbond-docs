Rainbond 内置了 Kubernetes Dashboard，可以通过 Kubernetes Dashboard 界面查看集群资源状态。

## 访问 K8s Dashboard

进入 **平台管理 -> 集群**，跳转到集群管理页面，点击 **集群监控**，进入 Kubernetes Dashboard 界面。
