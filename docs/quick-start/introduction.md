<!-- ## Rainbond是什么？ -->

Rainbond 是一款以应用为中心的开源容器平台。它底层基于 Kubernetes，但尽量屏蔽 Kubernetes 和 YAML 的复杂性，让团队用更低门槛完成应用的构建、部署、升级、运维和交付。

## 主要特点

- **不用先懂 Kubernetes**：底层基于 Kubernetes，但尽量屏蔽 YAML、集群和底层资源复杂性。

- **以应用为中心**：用统一的应用模型管理构建、部署、升级、回滚和运维。

- **多种部署方式**：支持源码、镜像、YAML、Helm 和软件包等常见方式。

- **持续交付与版本管理**：支持自动构建、持续发布、升级回滚和应用模板复用。

- **多集群与多云管理**：支持统一接入和管理多个 Kubernetes 集群及不同环境。

- **离线 / 信创 / 多架构支持**：适合客户现场、内网、国产化和 x86/ARM 混合场景。
