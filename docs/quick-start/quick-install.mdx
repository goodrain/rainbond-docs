本文将指引你快速安装一套可用的 Rainbond 环境，你可以根据自己的需求选择最适合你的安装方式。

:::info
- 刚了解 Rainbond ，想要快速上手的用户。我们推荐你选择[单机体验版安装](/docs/installation/install-with-dind)
- 对 Rainbond 已经有一定了解，但是对 Kubernetes 集群不熟悉的用户，我们推荐你选择[Web界面安装](/docs/installation/install-with-ui/host-install-with-ui)
- 对 Kubernetes 较为熟悉，想部署高可用环境的，我们推荐你选择[基于已有 Kubernetes 集群安装](/docs/quick-start/quick-install#基于已有-kubernetes-集群安装)
:::

## 基于已有主机安装

### [单机体验版安装](/docs/installation/install-with-dind)

了解在 Linux、Mac、Windows 下如何仅通过一个容器快速体验到 Rainbond 完整功能。仅适用于体验 Rainbond 功能或者个人开发环境。不可用于生产使用。

### [基于 Web 界面安装](/docs/installation/install-with-ui/host-install-with-ui)

了解在仅有 Linux 机器时，如何在图形化界面中安装 Rainbond 。这种安装方式相比单机体验版而言，需要用户手动部署 Kubernetes 集群和 Rainbond。复杂度会比单机体验版高， 且目前仅支持 Linux x86_64 机器。这种安装方式支持高可用部署。

## 基于已有 Kubernetes 集群安装

### [基于自建 Kubernetes 集群安装](/docs/installation/install-with-helm/k8s-install-with-helm/)

了解在自建的 Kubernetes 集群中，如何使用 Helm 快速部署 Rainbond 环境。这种安装方式支持高可用部署。

### [基于阿里云 ACK 集群安装](/docs/installation/install-with-helm/ack-install-with-helm/)

了解在阿里云 ACK 集群中，部署 Rainbond 的最佳实践，以及各类云服务资源的建议。这种安装方式支持高可用部署。

### [基于华为云 CCE 集群安装](/docs/installation/install-with-helm/cce-install-with-helm/)

了解在华为云 CCE 集群中，部署 Rainbond 的最佳实践，以及各类云服务资源的建议。这种安装方式支持高可用部署。

### [基于自建 k3s 集群安装](/docs/installation/install-with-helm/k3s-install-with-helm/)

了解在已有的 K3s 集群中，如何使用 Helm 快速部署 Rainbond 环境。这种安装方式支持高可用部署。

### [基于 Rancher 安装](/docs/installation/install-with-helm/install-from-rancher/)

了解在 Rancher 图形化界面中，如何使用应用商店快速部署 Rainbond 环境。
