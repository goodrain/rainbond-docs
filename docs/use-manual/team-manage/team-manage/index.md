## 修改团队名称

点击团队名称旁边的 `编辑` 按钮，填写新的团队名称并保存。

## 删除团队

点击 `删除团队` 按钮，即可删除团队。

:::danger
删除团队后，团队内的所有资源将被删除，请谨慎操作。
:::
