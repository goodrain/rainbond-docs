## 概述

企业基础插件是专为生产环境打造的核心功能扩展集合，旨在弥补开源版本在企业级应用场景下的功能缺口。

该插件包预置了六大关键能力模块：企业个性化定制、应用备份恢复、操作/登录日志审计、精细化权限管理、多语言运行时环境管理及资源超分配置。

## 功能对比

开源版本的 Rainbond 提供了基础的应用管理功能，但企业在实际生产环境中往往需要更高级的功能和定制化能力：

| 功能模块         | 开源社区版 (默认状态)                                        | 企业基础插件 (增强能力)                                      | 商业价值                                                     |
| ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 个性化定制       | 不支持自定义修改平台名称、Logo 及协议信息，界面固定。        | 支持企业品牌定制（Logo/名称）、界面个性化配置、自定义服务协议与平台相关设置。 | 品牌专属：打造符合企业形象的专属云管理平台，提升内部认同感与外部专业度。 |
| 应用备份恢复     | 不支持应用级的一键备份与还原机制。                           | 提供应用级的数据快照、备份策略及一键快速恢复能力。           | 业务连续：构建数据安全防线，确保在误操作或灾难场景下业务能快速复苏，降低停机损失。 |
| 操作/登录日志    | 不支持记录与查看详细的用户操作行为及登录历史，审计能力缺失。 | 提供完整的操作审计追踪、登录行为监控及日志导出功能，记录所有关键变更。 | 合规审计：满足企业安全合规要求，实现责任可追溯，快速定位异常行为。 |
| 精细化权限管理   | 仅提供基础的团队角色划分，缺乏细粒度的资源访问控制。         | 提供角色创建与修改，提供基于角色的细粒度权限控制（RBAC），支持精细化到应用甚至组件级别的权限管控。 | 安全管控：最小化权限原则，防止越权操作，全方位守护平台数据与架构安全。 |
| 多语言运行时环境 | 依赖平台默认环境                                             | 支持统一管理与分发多种语言的运行时环境（如 Java, Go, Node.js 等多版本），按需加载。 | 研发提效：统一构建标准，消除环境差异导致的构建失败，加速应用交付流程。 |
| 资源超分配置     | 严格按照物理资源限制分配，无法灵活调整资源利用率。           | 支持 CPU/内存资源的超分配置策略，允许根据业务负载特性灵活调整资源配额。 | 成本优化：最大化集群资源利用率，在保证业务稳定的前提下显著降低基础设施成本。 |


## 核心能力

### 企业个性化定制功能

让您的云管理平台独一无二！全方位的个性化配置，打造专属于您企业的品牌体验：

*   **品牌定制**：支持上传企业 Logo、网页图标 (Favicon)、登录页背景图
*   **界面配置**：可定制登录页标题、Slogan、页脚信息
*   **功能开关**：支持配置应用市场显示、官方 demo 展示、安全限制等
*   **服务协议**：可配置服务协议链接、隐私政策链接
*   **客服支持**：支持添加客服微信二维码，提供用户帮助入口

![image-20260228152034196](/plugins/base/individuation.png)

### 应用备份恢复功能

数据安全是企业的生命线！我们为您提供应用数据保护和恢复能力：

*   **一键备份**：支持应用数据的完整备份，包括组件数据、配置信息等
*   **备份管理**：提供备份列表管理、备份状态监控、备份导出功能
*   **恢复功能**：支持从备份中恢复应用数据，确保业务快速恢复

![image-20260228151750228](/plugins/base/backups.png)

### 操作/登录日志管理功能

运维排查，触手可及！统一管理平台的所有操作和访问日志：

*   **多维度日志**：覆盖操作日志和登录日志
*   **登录审计**：记录用户登录行为，包括登录时间、IP、设备信息和活跃时长
*   **操作记录**：记录所有用户的操作行为，包括应用部署、配置变更等
*   **查询筛选**：支持按用户、时间、操作类型等条件进行查询和筛选
*   **时长统计**：提供用户活跃时长统计，帮助了解平台使用情况

![image-20260228152244418](/plugins/base/log.png)

### 权限管理功能

精细化权限管理，让您的平台安全可控！谁能看什么，谁能做什么，一目了然：

*   **角色管理**：支持自定义角色，配置不同的权限组合
*   **用户权限**：可配置用户的平台访问权限、应用操作权限等
*   **资源隔离**：支持基于团队、应用的资源隔离和权限控制

![image-20260228152416084](/plugins/base/permission.png)

### 运行时环境管理功能

构建环境，随心所欲！统一管理各种开发语言和运行时环境的版本：

*   **多语言支持**：覆盖 Java、Node.js、Python、Golang、.NET 等主流开发语言
*   **版本管理**：提供完整的版本管理功能，支持设置默认版本、隐藏旧版本
*   **自动匹配**：构建过程中自动匹配项目所需的运行时环境版本

![image-20260228152503013](/plugins/base/language.png)

### 资源超分配置功能

资源优化，极致利用！灵活配置 CPU 和内存的超分比例，提升资源利用率：

*   **CPU 超分**：可配置 CPU 超分比例，最高支持 100% 超分
*   **内存超分**：可配置内存超分比例，最高支持 100% 超分
*   **边界控制**：设置超分范围限制，防止资源过度分配
*   **资源优化**：通过超分配置，实现资源的更高效利用

![image-20260228152536950](/plugins/base/excess.png)


## 使用指南

使用 企业基础插件，就是这么简单！完全融入 Rainbond 原生的管理体验：

1.  **访问插件中心**：在平台管理 -> 插件中心，查看所有可用的企业版插件
2.  **启用插件**：找到需要使用的插件，点击"启用"按钮
3.  **使用插件**：插件启用后，功能会自动集成到平台相应位置。
4.  **功能入口：**

| 功能                  | 功能入口                                               |
| --------------------- | ------------------------------------------------------ |
| 企业个性化定制功能    | 平台管理 -> 左侧导航栏（平台设置）-> 个性化配置        |
| 应用备份恢复功能      | 工作空间 -> 点击已创建应用 ->  左侧导航栏（应用备份）  |
| 操作/登录日志管理功能 | 平台管理 -> 左侧导航栏（平台日志）-> 操作日志/登陆日志 |
| 运行时环境管理功能    | 平台管理 -> 左侧导航栏（平台设置）-> 组件语言版本      |
| 权限管理功能          | 工作空间 -> 左侧导航栏（团队管理）->  角色             |
| 资源超分配置功能      | 平台管理 -> 左侧导航栏（平台设置）-> 超分比例          |
